# Маркетинговая Аналитика и Метрики

Этот каталог предназначен для анализа эффективности продаж, работы с CRM и оптимизации воронки.

## Структура
- **Scripts:**
  - `validate_csv.py`: Проверка структуры загружаемых CSV файлов (заголовки, телефоны).
  - `export_recovery_list.py`: Генерация списка на возврат (неоплаченные заказы).
  - `analyze_*.py`: Скрипты для анализа аномалий, CRM и месячной статистики.
  - `parse_tilda_stats.py`: Парсинг HTML отчетов Tilda.

## Инструкция для Агента
1. **Данные:** 
   - Ожидайте загрузки файлов `leads.csv` (выгрузка из Tilda) и `loyalty.csv` (база гостей) в эту папку.
   - Также сюда будут загружаться файлы статистики бизнеса.
   
2. **Валидация:**
   - Перед анализом всегда запускайте `python3 validate_csv.py` для проверки корректности данных.

3. **Задачи:**
   - Кросс-анализ "Брошенные корзины" vs "База лояльности".
   - Выявление системных проблем с оплатой (новые клиенты vs постоянные).
   - Генерация отчетов по эффективности каналов.
